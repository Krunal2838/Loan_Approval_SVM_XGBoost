# Loan Approval Prediction using SVM & XGBoost
The machine learning algorithm is being used in this project to predict whether a loan will be approved or not using the famous Loan Prediction Dataset. Two of the best models are implemented, analyzed, and compared to find out which one is better for the binary classification problem.

 An overview of the whole process involved in data science, including data loading and exploration, data preprocessing, model training, hyperparameter optimization, and performance evaluation.

## Dataset
The dataset is directly downloaded from a GitHub open source platform and consists of 614 rows and 13 columns in total. Some of these attributes are:
- **Demographics:** Gender, Marital status, Dependents, Education, Self-employed
- **Financials:** Applicant income, Coapplicant income, Loan amount, Loan amount term
- **Others:** Credit history, Property area
- **Target Variable:** Loan_Status (Y=approved, N=not approved)

The dataset includes many missing values in different columns.
